# Emoji Emotion Classifier using CNN and OpenCV

## Objective
Classify human facial expressions into emotions using a CNN model.

## Dataset
Facial emotion dataset with labeled expressions.

## Emotions Detected
- Happy
- Sad
- Angry
- Surprise
- Neutral

## Tools
Python, OpenCV, TensorFlow/Keras

## Steps
1. Image preprocessing
2. CNN model training
3. Emotion classification
4. Performance evaluation
